# Banned Vocabulary Reference

Compiled from: Wikipedia "Signs of AI Writing" editorial guide, GPTZero AI Vocabulary
analysis, Pangram Labs detection corpus, Embryo.com cross-model analysis (March 2025),
Max Planck Institute frequency study, and ETBI educator guide.

Read this file when using the human-writing skill. Never use any of these words or phrases
in output unless quoting someone else or the word has a literal technical meaning in context
(e.g., "leverage" in finance, "robust" in engineering).

## TIER 3: BANNED PHRASES AND PATTERNS

### Opening Cliches (never start anything with these)
- "In today's [adjective] [world/landscape/era/environment]..."
- "In the [fast-paced/rapidly evolving/ever-changing] world of..."
- "In an era where..."
- "In an increasingly [adjective] world..."
- "As we [navigate/enter/embrace] [the age of/a new era of]..."
- "As [field/technology] continues to evolve..."
- "[Topic] has [emerged as/become] a [game-changer/cornerstone/driving force]..."
- "The [rise/advent/emergence] of [X] has [transformed/revolutionized]..."
- "When it comes to [X]..."
- "Picture this:"
- "Imagine a world where..."
- "Have you ever wondered..."
- "Let's face it:"

### Mid-Text Padding
- "It's worth noting that..."
- "It's important to note that..."
- "It's important to remember that..."
- "It cannot be overstated..."
- "This is where [X] comes in."
- "This is particularly relevant because..."
- "This begs the question..."
- "Let's take a closer look."
- "Let's break this down."
- "Here's the thing:"
- "Here's where it gets interesting:"
- "The beauty of [X] is that..."
- "What sets [X] apart is..."
- "The bottom line is..."
- "At its core..."
- "Make no mistake..."
- "The reality is..."

### Closing Cliches
- "In conclusion..."
- "In summary..."
- "In essence..."
- "To sum up..."
- "All in all..."
- "At the end of the day..."
- "The future is [bright/promising/exciting]..."
- "Only time will tell..."
- "One thing is clear..."
- "The possibilities are [endless/limitless]..."
- "Moving forward..."
- "Going forward..."
- "As we look ahead..."

### Structural Patterns (the form, not specific words)
- "Not just X, but Y" / "It's not about X. It's about Y." (false contrast)
- "X, Y, and Z" where Z is a vague intensifier of Y (rule of three with padding)
- "While X, Y" / "Although X, it is Y" (hedging seesaw)
- "From X to Y" as a spectrum claim ("from startups to enterprises")
- "Whether you're [X] or [Y]" (false inclusivity)
- "[X] is more than just [Y]. It's [Z]." (inflated redefinition)
- "The question isn't [X]. It's [Y]." (false-depth reframe)

## TIER 4: CONTEXT-SPECIFIC TELLS

### Email/Professional Communication
- "I hope this email finds you well"
- "I hope this message finds you well"
- "Per our previous conversation"
- "Please do not hesitate to reach out"
- "I wanted to circle back"
- "Best regards" (overused; fine in genuinely formal contexts)
- "Thank you for your prompt response"
- "I appreciate your understanding"

### Marketing/Blog Copy
- "Unlock the [power/potential/secrets] of..."
- "The ultimate guide to..."
- "Everything you need to know about..."
- "[Number] [things/ways/tips/secrets] to..."
- "Take your [X] to the next level"
- "Ready to [verb]? Let's get started!"
- "Stop [bad thing]. Start [good thing]."
- "The secret to [X] that nobody talks about"

### Academic/Analytical
- "A comprehensive analysis"
- "This paper aims to..."
- "In this regard..."
- "With respect to..."
- "It bears mentioning..."
- "Scholars have long debated..."
- "A burgeoning field"
- "The extant literature suggests..."

## REPLACEMENT STRATEGY

Don't just delete banned words. Replace them with specific, concrete alternatives.

| Instead of | Try |
|---|---|
| utilize | use |
| facilitate | help, make possible, run |
| leverage | use, take advantage of, build on |
| optimize | improve, speed up, fix |
| implement | build, set up, start using |
| endeavor | try, attempt, work |
| comprehensive | full, complete, thorough, detailed |
| innovative | new, original, clever, different |
| robust | strong, solid, reliable, thorough |
| seamless | smooth, easy, simple |
| cutting-edge | new, modern, latest, current |
| streamline | simplify, speed up, clean up |
| enhance | improve, strengthen, add to |
| foster | encourage, support, build, grow |
| navigate | deal with, handle, figure out, work through |
| paradigm shift | change, shift, rethink |
| stakeholders | people involved, the team, customers, anyone affected |
| deliverables | what you'll ship, the output, the work |
| actionable insights | useful findings, things to act on |
| deep dive | close look, detailed review |

When tempted to use a fancy word, ask: would I say this out loud to a friend?
If not, use the word you would say out loud.
